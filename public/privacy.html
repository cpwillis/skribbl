Privacy & Storage Policy

Last updated: March 2026

Short version: On your first visit, Cloudflare Web Analytics collects anonymous, cookie-free page metrics. After that initial load the app is fully cached in your browser and can run completely offline - no further network requests and no further analytics. Everything else this app stores lives only in your own browser and never leaves your device. By using this site you accept this policy in full.

1. What this site is

Skribbl Solver is a free, open-source fan-made tool for players of skribbl.io. It is not affiliated with, endorsed by, or connected to skribbl.io or its operators. The source code is publicly available at github.com/cpwillis/skribbl.

2. Cloudflare Web Analytics

On the initial page load - and on any subsequent load where your browser fetches resources directly from the network rather than from the local cache - this site loads the Cloudflare Web Analytics beacon script from static.cloudflareinsights.com.

Cloudflare Web Analytics is a privacy-first service that:

  • Uses no cookies and sets no persistent identifiers.
  • Collects no personal data such as IP addresses, names, or email addresses.
  • Gathers only anonymous, aggregate metrics: page URL, referrer, approximate country, browser type, and device type.
  • Is governed by Cloudflare's Privacy Policy.

Once the service worker has cached the application (see §4), all app resources - HTML, CSS, JavaScript, and word lists - are served from your browser's local cache. The Cloudflare beacon is a cross-origin script that the service worker does not cache; it therefore only fires when your browser makes a live network request to this site. This site uses no other analytics service, tracking pixel, advertising script, or third-party cookie.

3. Browser storage (localStorage)

This site uses your browser's localStorage API - a standard browser feature - to save the following preferences locally on your device only:

  • Theme preference - whether you have selected light or dark mode.
  • Saved word list combinations - any named sets of word lists you choose to save using the "Save combo" feature.
  • Storage banner dismissed - whether you have dismissed the storage notice banner.

This data is classified as strictly necessary / functional and is used solely to improve your experience on this site. It is never sent to any server, never shared with any third party, and never used for advertising or analytics purposes.

Because this storage is functional and remains entirely on your device, it does not require opt-in consent under applicable privacy regulations. By continuing to use this site, you acknowledge use of browser storage for these purposes. You can clear this data at any time by clearing your browser's local storage for this site.

4. Service Worker & offline operation

On your first visit, this site installs a Service Worker that caches the entire application - HTML, CSS, JavaScript, and all word list data files - in your browser's cache storage. After this initial load the app is fully self-contained and runs completely offline. All subsequent visits are served from your local cache; no network connection is needed to use the app.

The Service Worker only caches same-origin resources. External scripts - including the Cloudflare Web Analytics beacon - are cross-origin and are therefore not cached. As a result, analytics will only fire when your browser makes a live network request to this site (i.e. before the cache is populated, or if the cache is cleared).

Cached data contains only the word list files included in this repository and is entirely local to your browser. You can inspect or remove this cache at any time via your browser's developer tools (Application → Cache Storage / Service Workers).
